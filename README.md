# alp-marketplace

Personal Claude Code plugin marketplace for [phucanh08](https://github.com/phucanh08).

## Install

```bash
/plugin marketplace add phucanh08/alp-marketplace
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [alp](https://github.com/phucanh08/alp-plugin) | Personal engineering plugin — skills, agents, workflows for daily development |

## Usage

After adding the marketplace, install any plugin:

```bash
/plugin install alp@phucanh08-alp-marketplace
```
